# Emotion Detection App

Welcome to the Emotion Detection App! This project is part of the IBM Generative AI Engineering Professional Certificate. 

## Features
- Analyzes text for emotions: **Anger**, **Disgust**, **Fear**, **Joy**, and **Sadness**.
- Powered by Watson NLP.
- Simple and intuitive interface built with Flask and HTML/JavaScript.

## How it Works
1. Enter your statement in the text box.
2. Click "Analyze".
3. The system returns the scores for each emotion and identifies the **dominant emotion**.

## Getting Started
To run this application, you'll need Python and Flask.
1. Clone the repository.
2. Install dependencies (requests, flask).
3. Run `python3 server.py`.
4. Open `http://localhost:5000` in your browser.

Enjoy analyzing!
